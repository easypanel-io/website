# NetAlertX

1-Click installation template for NetAlertX on Easypanel

## Description

NetAlertX is a network monitoring and notification framework that scans your Wi-Fi/LAN network and alerts you when new or unknown devices are detected. It provides real-time network device discovery, ARP scanning, and customizable alert notifications. Built for network administrators and security-conscious users, NetAlertX helps you maintain visibility over your network by tracking connected devices and identifying unauthorized access. The application features a web-based dashboard, GraphQL API, and supports various notification methods to keep you informed about network changes.

## Benefits

- Network Device Discovery: Automatically scan and discover all devices on your Wi-Fi/LAN network, providing complete visibility into connected devices.
- Real-Time Alerts: Receive immediate notifications when new or unknown devices join your network, helping you detect unauthorized access.
- Network Security Monitoring: Maintain awareness of all devices on your network to identify potential security threats or unauthorized connections.
- Self-Hosted Privacy: All network scanning and monitoring happens locally on your infrastructure, ensuring complete privacy and data control.
- Customizable Notifications: Configure alert notifications through various methods to stay informed about network changes.

## Features

- ARP Network Scanning: Advanced ARP scanning capabilities for comprehensive network device discovery and monitoring.
- Web Dashboard: Modern web-based interface for viewing network devices, managing alerts, and configuring monitoring settings.
- GraphQL API: Powerful GraphQL API for programmatic access to network data and integration with other tools.
- Device Tracking: Track known and unknown devices, maintain device history, and identify new connections in real-time.
- Customizable Alerts: Configure alert rules and notification methods to be notified when specific network events occur.
- Network Visualization: Visual representation of your network topology and connected devices for better understanding of network structure.
