# AriaNg

1-Click installation template for AriaNg on Easypanel

## Description

AriaNg is a modern web frontend for aria2, a lightweight multi-protocol &amp; multi-source command-line download utility. It provides a beautiful web interface for managing downloads.

## Benefits

- Modern Interface: Beautiful and responsive web interface
- Multi-protocol: Support for HTTP, HTTPS, FTP, SFTP, BitTorrent, and more
- Remote Control: Control downloads remotely through RPC
- Open Source: Free and open-source download manager

## Features

- Web Interface: Modern and responsive web UI
- Multiple Protocols: Support for various download protocols
- Remote Access: Access and control downloads remotely
- Download Management: Pause, resume, and manage downloads
- Configuration: Customizable download settings
