1-Click installation template for GoPeed on Easypanel

## Description

GoPeed is a high-performance download manager that supports HTTP, HTTPS, FTP, BitTorrent, and other protocols. It features a modern web interface and supports multiple download task.

## Benefits

- High Performance: Fast and efficient download management with support for multiple protocols
- Modern Interface: Clean and intuitive web-based user interface
- Multi-protocol Support: Download from HTTP, HTTPS, FTP, BitTorrent, and more
- Open Source: Free and open-source platform with active development

## Features

- Web Interface: Access and manage downloads through a modern web UI
- Multiple Protocols: Support for various download protocols
- Download Management: Pause, resume, and manage multiple download tasks
- Progress Tracking: Real-time download progress monitoring
- Custom Storage: Configurable download storage location
