# Viseron

1-Click installation template for Viseron on Easypanel

## Description

Viseron is a self-hosted, local network video recorder (NVR) and AI-powered video surveillance system designed for privacy-conscious users. It supports multiple camera streams with advanced features like object detection, motion detection, face recognition, and license plate recognition using machine learning models. Viseron can record continuous video, create event-based clips, generate timelapses, and send real-time notifications. It integrates seamlessly with Home Assistant and supports various camera protocols including RTSP, HTTP, and USB cameras. The system features a modern web interface for live viewing, playback, and configuration. All processing happens locally on your hardware, ensuring your video data never leaves your network. Perfect for home security, surveillance systems, and smart home automation.

## Benefits

- Privacy-First Design: All video processing and AI analysis happens locally on your hardware. Your video footage never leaves your network, ensuring complete privacy and security.
- AI-Powered Detection: Advanced object detection, motion detection, face recognition, and license plate recognition using state-of-the-art machine learning models.
- Home Assistant Integration: Native integration with Home Assistant for seamless smart home automation, allowing you to trigger actions based on detected events.

## Features

- Multi-Camera Support: Monitor multiple camera streams simultaneously with support for RTSP, HTTP, and USB cameras. Configure each camera independently.
- Object & Motion Detection: Intelligent object detection with customizable zones and filters. Detect people, vehicles, animals, and more with high accuracy.
- Event Recording & Clips: Automatically record event-based clips when motion or objects are detected. Configure pre and post-recording buffers for complete context.
- Timelapse & Continuous Recording: Create beautiful timelapses and maintain continuous recording segments for comprehensive surveillance coverage.
- Real-Time Notifications: Receive instant notifications with snapshots and thumbnails when events are detected via multiple notification channels.
- Web Interface: Modern, responsive web interface for live viewing, playback, event browsing, and system configuration accessible from any device.
