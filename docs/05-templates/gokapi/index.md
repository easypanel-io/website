# Gokapi

1-Click installation template for Gokapi on Easypanel

## Description

Gokapi is a lightweight file-sharing server that allows you to easily upload and share files with others. It provides a simple web interface for file uploads, generates shareable links, and supports features like password protection, automatic file expiration, and customizable download limits.

## Benefits

- Simple File Sharing: Easily share files with others through a clean and intuitive web interface.
- Enhanced Privacy Controls: Set passwords, expiration dates, and download limits for each shared file.
- Self-Hosted Solution: Keep control of your data by hosting the file sharing service on your own infrastructure.

## Features

- Upload Management: Upload files through a web interface and generate shareable links instantly.
- Access Controls: Protect shared files with passwords and set automatic expiration dates.
- Download Limits: Limit the number of times a file can be downloaded before it expires.
