# SFTPGo

1-Click installation template for SFTPGo on Easypanel

## Description

SFTPGo is a fully featured and highly configurable SFTP server with optional HTTP/S, FTP/S and WebDAV support. It supports local filesystem, encrypted local filesystem, S3 (compatible) Object Storage, Google Cloud Storage, Azure Blob Storage, SFTP.

## Benefits

- Multiple Protocols: Support for SFTP, HTTP/S, FTP/S, and WebDAV protocols
- Multiple Storage Backends: Support for local filesystem, S3, Google Cloud Storage, and more
- Security Features: Built-in security features and encryption support
- Open Source: Free and open-source platform with active community support

## Features

- Web Interface: User-friendly web interface for management
- User Management: Comprehensive user and permission management
- Virtual Folders: Support for virtual folders and path mapping
- Quota Management: Set storage quotas for users and groups
- Event Notifications: Configure notifications for various events
- Backup Support: Built-in backup and restore functionality
