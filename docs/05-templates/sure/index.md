# Sure

1-Click installation template for Sure on Easypanel

## Description

Sure is a self-hosted personal finance management application built with Ruby on Rails. It helps individuals and families track expenses, manage budgets, monitor income, and gain insights into their financial health. Sure features an intuitive interface for recording transactions, categorizing expenses, creating budgets, and generating financial reports. With real-time data processing via Sidekiq background jobs and optional AI-powered features through OpenAI integration, Sure provides intelligent financial insights, spending pattern analysis, and automated categorization. Built for privacy-conscious users, Sure keeps all your financial data on your own infrastructure without sharing sensitive information with third-party services. Perfect for anyone seeking complete control over their personal finance tracking and analysis.

## Benefits

- Complete Financial Privacy: Keep all your financial data on your own infrastructure. Your sensitive banking information, transactions, and spending habits never leave your server.
- AI-Powered Insights: Optional OpenAI integration provides intelligent spending analysis, automated transaction categorization, and personalized financial recommendations.
- Real-Time Financial Tracking: Background job processing with Sidekiq ensures instant transaction updates, budget calculations, and financial report generation.

## Features

- Expense Tracking: Record and categorize all your expenses with detailed transaction history, receipts, and notes for comprehensive financial tracking.
- Budget Management: Create custom budgets for different categories, set spending limits, and receive alerts when approaching or exceeding budget thresholds.
- Income Monitoring: Track multiple income sources, monitor cash flow, and analyze earning patterns over time with detailed reporting.
- Financial Reports: Generate comprehensive financial reports including spending summaries, category breakdowns, trends analysis, and custom date ranges.
- Automated Categorization: Use AI-powered transaction categorization to automatically organize expenses and income with smart pattern recognition.
- Multi-Currency Support: Track finances in multiple currencies with automatic conversion and support for international transactions.
